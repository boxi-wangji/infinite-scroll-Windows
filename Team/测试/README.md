# 测试职责说明

## 角色定位

负责功能验收测试、会话恢复专项测试、快捷键全覆盖测试及回归测试。

---

## 测试环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 21H2+ 或 Windows 11 |
| Node.js | 18.x LTS 或 20.x LTS（dev 环境） |
| 安装包测试 | 全新 Windows 环境（无 Node.js / Electron） |
| 测试 Shell | PowerShell 7、PowerShell 5.1、CMD 均需覆盖 |

---

## 测试模块

### 1. 终端基础功能
- [ ] 终端能正常渲染、输入、输出
- [ ] ANSI 颜色（Dracula 配色）正常显示，无紫色目录名
- [ ] 中文字符不乱码
- [ ] 终端大小随窗口 / 面板自适应调整

### 2. 无限滚动画布
- [ ] 新增行（Ctrl+Shift+↓）：行数累加，画布可向下滚动
- [ ] 单面板时撑满窗口；多面板时固定高度可滚动
- [ ] 拖动行间分割线调整高度：流畅，松手后重启保留高度

### 3. 快捷键全覆盖

| 快捷键 | 预期行为 |
|--------|----------|
| Ctrl+Shift+↓ | 新增行 |
| Ctrl+D | 复制当前面板 |
| Ctrl+W | 关闭当前面板 |
| Ctrl+↑/↓ | 焦点上移/下移 |
| Ctrl+←/→ | 焦点左移/右移 |
| Ctrl+= / Ctrl+- | 字体放大/缩小 |
| Ctrl+滚轮 | 字体缩放（终端内外均生效） |
| Ctrl+C（有选中） | 复制到剪贴板，不发中断 |
| Ctrl+C（无选中） | 发中断信号 ^C |
| Ctrl+V | 粘贴剪贴板，只粘贴一次（不重复） |
| Ctrl+/ | 打开/关闭帮助面板 |

### 4. 右键菜单
- [ ] 终端内容区右键弹出菜单
- [ ] 面板头部右键弹出菜单
- [ ] 复制：有选中文字时生效，无选中时标签变灰
- [ ] 粘贴：内容正确写入终端，不重复
- [ ] 清屏：终端清空
- [ ] 复制面板 / 关闭面板：功能正常
- [ ] 点击菜单外或按 Esc 关闭菜单

### 5. 标签重命名
- [ ] 双击标签名进入编辑状态
- [ ] Enter 确认，Esc 取消
- [ ] 重命名后重启仍保留

### 6. Shell 切换
- [ ] 头部下拉框显示已安装的 Shell（PS7 / PS5 / CMD）
- [ ] 切换后终端重启，使用新 Shell
- [ ] 新 Shell 的颜色 / 提示符正常显示
- [ ] 切换 Shell 后重启，面板仍使用切换后的 Shell

### 7. 会话持久化（方式 B 专项）
- [ ] 正常关闭后重启：布局（行数、列数、高度、标签名、Shell）恢复
- [ ] cwd 恢复：新 Shell 在原目录打开
- [ ] scrollback 恢复：历史输出可见，有 `[Session restored]` 标记
- [ ] 强制终止进程后重启：布局恢复，scrollback 可能丢失（可接受）

### 8. 笔记面板
- [ ] 内容输入、保存、重启后读取

### 9. 安装包测试
- [ ] NSIS 安装包在全新环境安装，无报错
- [ ] 桌面快捷方式、开始菜单正常创建
- [ ] 应用图标正常显示（任务栏 / 桌面 / 标题栏）
- [ ] 卸载后无残留文件（%APPDATA% 配置除外）

## 已知 Edge Cases

| 场景 | 预期行为 |
|------|----------|
| 终端进程被外部 kill | `pty:exit` 触发，面板显示"已断开"，不崩溃 |
| 关闭最后一个 cell | 自动补一个新面板，不空白 |
| 字体超出范围（<8 或 >24） | 夹在边界值 |
| 窗口极小（<200px） | FitAddon 不崩溃，min cols=1 rows=1 |
| 快速连续新增多行 | 不卡顿，不重复创建 |
| Ctrl+V 键盘粘贴 | 只写入一次（xterm 原生处理，不重复） |
| Shell 路径不存在 | ptyManager fallback，终端显示错误提示 |
